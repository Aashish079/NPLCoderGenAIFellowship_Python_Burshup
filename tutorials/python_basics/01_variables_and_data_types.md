# Variables and Data Types

This tutorial covers the basics of variables and data types in Python.

## Variables

In Python, you can create a variable by assigning a value to it:

```python
x = 5
y = "Hello, World!"
```

## Data Types

Python has several built-in data types. Here are some of the most common ones:

*   **Text Type:** `str`
*   **Numeric Types:** `int`, `float`, `complex`
*   **Sequence Types:** `list`, `tuple`, `range`
*   **Mapping Type:** `dict`
*   **Set Types:** `set`, `frozenset`
*   **Boolean Type:** `bool`
*   **Binary Types:** `bytes`, `bytearray`, `memoryview`
